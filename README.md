### quickstart
- start keycloak+mssql > `docker-compose up`
- start host (Api+ClientApp) > `Visual Studio CTRL+F5`
- browse url > `http//localhost:5001`
- login > `admin/admin`


### management
- keycloak > `http://localhost:8080/auth/admin` 
- login > `admin/admin`
- clients > `demo`
