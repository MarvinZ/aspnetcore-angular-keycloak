### quickstart
- start keycloak+mssql > `docker-compose up`
- start host (api+ng) > `Visual Studio CTRL+F5`
- browse url > http://localhost:5001 (admin/admin)

### showcase

- angular 10

- generated ts api client
- integration tests (+FakeAuthenticationHandler)

### management
